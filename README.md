### getArea() 함수 사용법

첫 번째 인자가 다각형의 형태를 결정한다.  

* 'circle' : 원
* 'rect' : 사각형
* 'trapezoid' : 사다리꼴

두 번째 인자부터 각 다각형 함수에 맞는 인자를 넣어준다.  

* 원
1. 인자가 하나인 경우 : 반지름을 넣어준다
2. 인자가 두 개인 경우 : 시작 반지름부터 끝 반지름까지 반지름이 1씩 커지는 원의 넓이를 모두 구한다

* 사각형 : 인자를 두 개 받아(밑변, 높이) 넓이를 출력한다.

* 인자를 세 개 받아(밑변, 윗변, 높이) 넓이를 출력한다.
